# React Native Fitness App

A fitness app built using Expo, React Native, and React Navigation.

## Demo

![React Native Fitness App Demo](./readme-images/React-Native-Fitness-App.png)

## Get Started

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm start
```
